Why web development?

I am not a full-fledged web developer. I learned by myself how to make a site, used promming / markup languages like php, javascript, HTML/CSS and frameworks like bootstrap.

I noticed that I didn't like my former job anymore. There weren't enough possibilities to grow. I had to reinvent myself.

I really liked making websites, but I experienced that I needed some more depth to the products I was developing and was missing focus. I was doing all kinds of things like making a design, design a logo in Illustrator, copywriting, using coding languages like php, javascript.

While I was building these sites, all kinds of questions came into my head which I did not have an answer for. How do I handle SEO? How do I intete different systems by using an API? How do I speed up the page load.

As you can see, I had too many questions. With every question a new one popped up. The biggest question remain.. Where do I start?

It was time to take some action. I was looking for a good course to advance in web development. I started a course at freecodecamp and asked for advice from developers I know. I also experienced that there are good courses on the web. By chance, I found codaisseur’s website. I really liked what they offered. It’s a good curriculum and with a clear focus.
